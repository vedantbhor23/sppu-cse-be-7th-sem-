# Advanced Titanic Survival Prediction using Machine Learning

## Overview

This project presents a comprehensive machine learning pipeline for predicting Titanic passenger survival. It includes data preprocessing, feature engineering, model training, evaluation, hyperparameter tuning, and prediction generation.

## Features

- Data Cleaning
- Feature Engineering
- Missing Value Handling
- Multiple Machine Learning Models
- Cross Validation
- Model Evaluation
- SHAP Explainability
- Prediction Submission Generation

## Algorithms Used

- Logistic Regression
- Random Forest
- XGBoost
- Decision Trees

## Technologies Used

- Python
- Pandas
- NumPy
- Scikit-learn
- XGBoost
- Matplotlib
- SHAP
