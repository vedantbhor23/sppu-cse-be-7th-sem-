# Multithreaded Merge Sort Visualization

## Overview

This project demonstrates the implementation of Merge Sort and Multithreaded Merge Sort with graphical visualization. It compares execution times and showcases the performance benefits of parallel processing.

## Features

- Merge Sort Implementation
- Multithreaded Merge Sort
- Performance Comparison
- Random Array Generation
- Graphical Visualization

## Technologies Used

- Python
- Tkinter
- threading
- time
